# VibeVerse

VibeVerse is now a Medium-style blog writing website built with vanilla HTML/CSS/JS.

## Features

- Clean publication-style feed layout
- Search by title, author, tag, or content
- Sorting tabs: For you / Popular / Recent
- Story publishing form with title, author, tags, and content
- Local persistence using `localStorage`
- Clap and delete interactions for each story

## Run locally

Open `index.html` in a browser.
